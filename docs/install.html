Source

The source code for Night can be found on the GitHub page.

Step 1: Install Dependencies

You'll need the following:

  • g++-7 or clang-5
  • make
  • cmake

Step 2: Clone Repository

Make sure to do a recursive clone to fetch the submodules as well.

git clone --recurse-submodules //github.com/dynamicsquid/night.git
cd night

Step 3: Build Code

mkdir build
cd build

cmake ../
make
